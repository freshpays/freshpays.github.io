FreshPays

The Bitpay client for Freshbooks

FreshPays is a Bitpay client, that allows you to accept Bitcoin payments on your FreshBook invoices.

FreshBooks Plugin

This application is a plugin for FreshBooks, the online invoicing service.

Bitpay Client

In order to receive Bitcoin payments, you need to sign up for Bitpay.

Open Source

This plugin was written as an open-source project, hosted on GitHub.
